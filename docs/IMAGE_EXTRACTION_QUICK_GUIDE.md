# 图片提取快速参考卡

## 核心原则

1. **只提取计算建模/模拟相关的图片**
2. **图片说明使用中文（保留原文编号）**
3. **多个图片创建多条记录（数组格式）**

---

## 图片筛选规则

### ✅ 必须提取（计算模拟相关）

```
✅ 有限元分析（FEA）结果图
✅ 应力分布云图（Von Mises、主应力等）
✅ 位移/变形分布图
✅ 接触压力分布图
✅ 应变分布图
✅ 有限元模型网格图
✅ 温度场分布图
✅ 磨损深度预测图（模拟）
```

### ❌ 不要提取（实验/照片）

```
❌ 材料微观结构照片（SEM、金相）
❌ 实验装置照片
❌ 产品实物照片
❌ X光片、CT扫描图
❌ 磨损形貌照片（实验结果）
❌ 材料表面形貌图
❌ 流程图、示意图
❌ 纯几何模型（无分析结果）
```

### 判断标准

| 类型 | 特征 | 是否提取 |
|------|------|----------|
| **模拟结果** | 有云图、应力值、网格 | ✅ 提取 |
| **实验照片** | 拍摄、显微镜观察 | ❌ 不提取 |
| **不确定** | 无法判断 | 留空（null） |

---

## 图片说明语言规范

### 推荐格式（中文为主）

```
✅ "图3 有限元模型网格划分，采用四面体单元，节点数12580个"
✅ "Fig. 2 Von Mises应力分布云图，最大应力85 MPa出现在内衬边缘"
✅ "图4-1 接触压力分布。峰值压力为50 MPa，位于承重区"
```

### 避免的格式

```
❌ "Fig. 2 Stress distribution"（全英文）
❌ "应力分佈雲圖"（繁体字）
❌ "应力分布图"（缺少图号和关键信息）
```

### 保留项

- ✅ 图片编号（Fig. X、图X-X）
- ✅ 专业术语缩写（Von Mises、FEA）
- ✅ 标准单位（MPa、mm、GPa）

## 检查清单

提取前检查：
- [ ] 图片是否为计算模拟结果？
- [ ] 图片是否包含云图或网格？
- [ ] 图片是否有应力/位移等数值？

提取后检查：
- [ ] 路径格式正确？（images/xxx.jpg）
- [ ] 说明使用中文？
- [ ] 保留了图片编号？
- [ ] 包含了关键数值？
- [ ] 实验照片是否被正确排除？

---

## 快速决策流程图

```
图片 → 是否有云图/网格/应力值？
       ├─ 是 → ✅ 提取（模拟结果）
       └─ 否 → 是否为实验拍摄？
               ├─ 是 → ❌ 不提取
               └─ 否 → 不确定 → null
```

---

## 记住

1. **宁少勿多**：不确定的不要提取
2. **中文优先**：说明用中文，更易读
3. **只要模拟**：实验照片不要提取到此表
4. **保留编号**：图片编号保留原文
