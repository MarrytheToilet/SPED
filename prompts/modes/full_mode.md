# 全量提取模式 - 单次完整提取

## 🎯 任务目标

从完整论文中一次性提取所有人工关节材料实验数据，输出完整的12表JSON结构。

> 此模式适用于内容较短的论文（<8000字符），可以在一次LLM调用中完成提取。

## 📋 提取步骤

### 第一步：识别独立记录
- 不同材料/处理 → 不同记录
- 同一材料的多个指标 → 同一记录

### 第二步：提取每条记录的数据
- 遍历12张表的所有字段
- 找到数据就填，找不到填 `null`

### 第三步：检查输出
- 确保JSON格式正确
- 确保数据准确无误

---

## 🚫 提取规则（必须遵守）

1. **禁止编造** - 论文没写的填 `null`
2. **禁止转换** - `2.8 cm` 不要转成 `28 mm`
3. **保留精度** - `0.16±0.02` 原样保留
4. **DOI格式** - 必须以 `10.` 开头，否则填 `null`
