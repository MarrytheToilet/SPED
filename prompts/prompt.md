# 人工关节材料数据提取 - 严格规范版

## 📋 任务目标

从学术论文中提取**人工关节（髋关节、膝关节等）材料的实验数据**，按照规范化多表结构输出JSON。

---

## ⚠️ 核心原则：宁缺毋滥

### 黄金规则
1. **只提取明确的信息** - 论文中明确写出的数据
2. **不推测、不猜测** - 不确定的一律留空（null）
3. **不编造、不补全** - 缺失的数据不要尝试填充
4. **保持中立** - 不做任何解释或推断

### 什么时候留空（null）
- ❌ 论文未明确提及该字段
- ❌ 信息模糊或不确定（如"约30mm"可以提取为"约30 mm"，但"大约3cm左右"太模糊则留null）
- ❌ 只有范围值但不是精确实验数据（如"硬度通常在400-500之间"）
- ❌ 综述中的概括性描述（如"磨损率一般在10⁻⁶数量级"）
- ❌ 无法确认单位的数值（必须有明确单位）

### 什么时候提取
- ✅ 有明确数值 + 明确单位
- ✅ 实验论文中作者的实验数据
- ✅ 综述论文中引用的具体实验数据（需标注来源）
- ✅ 明确的材料名称、工艺描述
- ✅ 明确的测试条件和参数

---

## 🎯 论文类型识别与处理策略

### 类型1: 实验论文（Experimental Paper）
**特征**: 作者进行了自己的实验，报告完整的材料、方法、结果
**处理**: 提取所有实验数据，创建完整记录

### 类型2: 人工关节综述（Review Paper - 相关领域）
**特征**: 
- 标题包含"进展"、"综述"、"Review"、"Progress"
- 涉及人工关节材料和实验
- 引用多篇文献的实验数据

**处理策略** ⚠️:
- ✅ **提取引用的具体实验数据**（有明确数值、材料组合、测试条件）
- ❌ **不提取**概括性描述（如"磨损率在10⁻⁶范围"）
- ✅ **每个引用实验**创建一条独立记录
- ✅ 在"产品所属专利号或文献"字段注明原始来源

**示例**:
```text
论文中提到："文[15]研究表明，CoCrMo球头(Ø28mm)与UHMWPE内衬
(厚度6.3mm)配合，在100万次循环后磨损率为2.3 mg/Mc"
```
✅ **应该提取**，并标注来源为"引用文献[15]"

### 类型3: 其他领域综述（Review Paper - 不相关）
**特征**: 讨论普通材料、其他医疗器械、非关节领域
**处理**: 返回 `{"records": []}`

## 🔍 数据提取规则（重要⚠️）

### 1. 精确性原则
- ✅ 只提取**明确的数值、材料名称、测试条件**
- ❌ 不推测、不编造、不提取模糊描述
- ❌ 避免"范围值"（如10⁻⁶~10⁻⁷），除非论文就是这样报告的

### 2. 单位强制要求
**所有数值必须带单位**，格式：`数字 + 空格 + 单位`

| 类型 | 单位 | 示例 |
|------|------|------|
| 直径、长度 | mm | `28 mm`, `6.3 mm` |
| 表面粗糙度（球头） | nm | `50 nm`, `100 nm` |
| 表面粗糙度（内衬/股骨柄） | μm | `0.1 μm`, `0.5 μm` |
| 硬度 | HV, HRC, HB | `450 HV`, `60 HRC` |
| 强度 | MPa, GPa | `500 MPa`, `2.5 GPa` |
| 密度 | g/cm³ | `8.3 g/cm³` |
| 磨损率 | mg/Mc, mm³/Nm | `2.3 mg/Mc`, `1.5×10⁻⁶ mm³/Nm` |

### 3. 材料名称规范化

| 原始名称 | 规范名称 |
|----------|----------|
| 超高分子量聚乙烯、UHMWPE | UHMWPE |
| 交联聚乙烯、XLPE、X-linked PE | XLPE |
| 钴铬钼合金、CoCrMo、Co-Cr-Mo | CoCrMo |
| 钛合金、Ti6Al4V、Ti-6Al-4V | Ti-6Al-4V |
| 氧化锆陶瓷、ZrO2、Zirconia | ZrO₂陶瓷 |
| 氧化铝陶瓷、Al2O3、Alumina | Al₂O₃陶瓷 |

### 4. 独立记录创建规则

每个独立实验创建一条记录，判断标准：
- ✅ 不同材料组合（如CoCrMo-UHMWPE vs 陶瓷-XLPE）
- ✅ 不同测试条件（如不同载荷、不同周期）
- ✅ 不同实验批次或样本组
- ✅ 综述中引用的不同文献实验

### 5. 空值处理（关键⚠️）

#### 字段级空值
**必须使用 `null`，不要使用其他任何值**

❌ 错误示例：
```json
"内衬厚度(mm)": "",          // 错误：空字符串
"内衬厚度(mm)": "未提及",    // 错误：文字说明
"内衬厚度(mm)": "无",        // 错误：文字说明
"内衬厚度(mm)": "N/A",       // 错误：缩写
"内衬厚度(mm)": "不清楚",    // 错误：文字说明
```

✅ 正确示例：
```json
"内衬厚度(mm)": null,        // 正确：使用null
"内衬厚度(mm)": "6.3 mm",    // 正确：有值时提供完整信息
```

#### 表级空值
整个表无任何数据时，使用 `null`：

✅ 正确示例：
```json
{
  "基本信息表": {...},
  "内衬基本信息表": {...},
  "球头基本信息表": {...},
  "配合信息表": null,          // 正确：整个表无数据
  "股骨柄基本信息表": null,    // 正确：整个表无数据
  ...
}
```

#### 常见字段的空值判断

| 字段 | 留空（null）的情况 | 可以提取的情况 |
|------|-------------------|---------------|
| **尺寸类** | "厚度约6mm左右" | "厚度6.3 mm"、"厚度6±0.2 mm" |
| **硬度类** | "硬度较高" | "硬度450 HV"、"硬度>400 HV" |
| **粗糙度** | "抛光处理" | "Ra<50 nm"、"Ra 0.05 μm" |
| **材料** | "高分子材料" | "UHMWPE"、"交联聚乙烯" |
| **工艺** | "经过特殊处理" | "100 kGy辐照交联" |
| **测试条件** | "标准测试条件" | "ISO 14242-1, 载荷2000 N" |

## ⚠️ 关键规则总结

### 1. 相关性判断
- ✅ **人工关节相关**: 髋关节、膝关节、肩关节等假体材料
- ✅ **人工关节综述**: 提取引用的具体实验数据
- ❌ **不相关**: 普通材料研究、其他医疗器械

### 2. 综述论文处理（新增⚠️）
- ✅ 提取**引用文献的具体实验数据**
- ✅ 每个引用实验创建**独立记录**
- ✅ 在"产品所属专利号或文献"注明**来源**（如"引用文献[15]"）
- ❌ 不提取**概括性描述**（如"磨损率通常在10⁻⁶范围"）
- ❌ 不提取**范围值**（除非原文就是这样）

### 3. 数据ID规则
- 格式: `AJ_YYYYMMDD_随机字符串`
- 同一记录所有表使用**相同ID**
- 不同实验/引用使用**不同ID**

### 4. 单位规范（关键⚠️）
| 项目 | 单位 | 格式 |
|------|------|------|
| 长度 | mm | `28 mm` |
| 球头粗糙度 | **nm** | `50 nm`, `Ra<50 nm` |
| 其他粗糙度 | **μm** | `0.1 μm`, `Ra 0.1 μm` |
| 硬度 | HV/HRC/HB | `450 HV` |
| 强度 | MPa/GPa | `500 MPa` |
| 密度 | g/cm³ | `8.3 g/cm³` |
| 磨损率 | mg/Mc, mm³/Nm | `2.3 mg/Mc`, `1.5×10⁻⁶ mm³/Nm` |

### 5. 材料名称规范化
使用规范名称：`UHMWPE`, `XLPE`, `CoCrMo`, `Ti-6Al-4V`, `ZrO₂陶瓷`, `Al₂O₃陶瓷`

### 6. 空值处理
- 整表无数据: `null`
- 字段无值: `null`
- ❌ 不用 `""`, `"未提及"`, `"无"`

## ❌ 常见错误示例（反面教材）

### 错误1：使用错误的空值标记

❌ **错误**:
```json
{
  "内衬厚度(mm)": "未提及",
  "球头直径(mm)": "",
  "内衬加工工艺": "无",
  "球头后处理": "N/A"
}
```

✅ **正确**:
```json
{
  "内衬厚度(mm)": null,
  "球头直径(mm)": null,
  "内衬加工工艺": null,
  "球头后处理": null
}
```

---

### 错误2：提取模糊或概括性描述

**论文原文**: "内衬采用高分子材料制成，经过特殊表面处理"

❌ **错误**:
```json
{
  "内衬材料名称": "高分子材料",
  "内衬后处理": "特殊表面处理"
}
```

✅ **正确**:
```json
{
  "内衬材料名称": null,  // "高分子材料"太模糊
  "内衬后处理": null     // "特殊表面处理"不明确
}
```

---

### 错误3：提取范围值（非实验数据）

**论文原文**: "CoCrMo球头的硬度通常在400-500 HV之间"

❌ **错误**:
```json
{
  "球头硬度(HV)": "400-500 HV"  // 这是概括性描述，不是实验数据
}
```

✅ **正确**:
```json
{
  "球头硬度(HV)": null  // 不是具体实验测量值
}
```

**但如果是实验报告**: "测试样品1硬度450 HV，样品2硬度480 HV，范围450-480 HV"

✅ **这种情况可以提取**:
```json
{
  "球头硬度(HV)": "450 HV, 480 HV (范围450-480 HV)"
}
```

---

### 错误4：缺少单位或单位错误

❌ **错误**:
```json
{
  "球头直径(mm)": "28",           // 缺少单位
  "内衬厚度(mm)": "6.3mm",        // 没有空格
  "球头表面粗糙度(nm)": "50 μm"   // 单位错误（应该是nm）
}
```

✅ **正确**:
```json
{
  "球头直径(mm)": "28 mm",
  "内衬厚度(mm)": "6.3 mm",
  "球头表面粗糙度(nm)": "50 nm"
}
```

---

### 错误5：材料名称不规范

❌ **错误**:
```json
{
  "内衬材料名称": "超高分子量聚乙烯",
  "球头材料名称": "钴铬钼合金",
  "内衬材料名称": "Al2O3"
}
```

✅ **正确**:
```json
{
  "内衬材料名称": "UHMWPE",
  "球头材料名称": "CoCrMo",
  "内衬材料名称": "Al₂O₃陶瓷"
}
```

### 错误6：推测或补全数据

**论文原文**: "使用标准的28mm球头"

❌ **错误**（不要推测其他信息）:
```json
{
  "球头直径(mm)": "28 mm",
  "球头材料名称": "CoCrMo",      // ❌ 论文未提及，不要推测
  "球头表面粗糙度(nm)": "Ra<50 nm" // ❌ 论文未提及，不要推测
}
```

✅ **正确**:
```json
{

  "球头材料名称": null,           // 论文未明确提及
  "球头表面粗糙度(nm)": null      // 论文未明确提及
}
```

### 错误7：综述论文处理不当

**论文原文**: "文献[15]研究表明，CoCrMo-UHMWPE的磨损率约为2 mg/Mc"

❌ **错误**（缺少来源标注）:
```json
{
  "基本信息表": {
    "产品所属专利号或文献": "综述论文自身的DOI"  // ❌ 错误
  },
  "性能测试结果表": {
    "磨损率": "约2 mg/Mc"  // ❌ "约"表示不确定
  }
}
```

✅ **正确**:
```json
{
  "基本信息表": {
    "产品所属专利号或文献": "引用文献[15]"  // ✅ 标注原始来源
  },
  "性能测试结果表": {
    "磨损率": "2 mg/Mc"  // ✅ 如果原文确实有"约"，可以保留："约2 mg/Mc"
  }
}
```

### 错误8：不该提取的内容

**论文原文**: "CoCrMo合金因其优异的耐磨性和耐腐蚀性而被广泛应用于人工关节"

❌ **错误**（提取了概括性描述）:

```json
  }
```

```json
{
  "球头材料名称": "CoCrMo",
  "球头物理性能表": null  // 没有具体性能数据
}
```

2. **原文为准** - 只提取论文明确写出的
3. **格式统一** - 空值统一用 `null`
4. **单位必带** - 所有数值必须有单位
5. **名称规范** - 使用标准材料名称
6. **来源明确** - 综述要标注引用来源
7. **独立记录** - 不同实验分开记录
