# 人工关节材料数据提取

## 任务目标

从学术论文中提取人工关节材料的实验数据，输出结构化JSON。

## 核心规范

1. **精确提取** - 只提取明确提到的信息，不推测、不编造
2. **数值带单位** - 所有数值必须包含单位，如 "450 HV"、"28 mm"、"0.05 μm"
3. **JSON格式** - 复杂信息用JSON字符串表示（内部引号必须转义 `\"`）
4. **字段完整** - 必须包含所有28个字段，未提及的填 `null`
5. **独立记录** - 不同材料组合/实验条件/测试批次各创建一条记录

## 数据Schema

{SCHEMA}

## 输出格式

### Full模式（全文提取）
```json
{
  "records": [
    {
      "数据标识": "...",
      "应用部位": "...",
      ...全部28个字段
    },
    {
      "数据标识": "...",
      ...第二条记录
    }
  ]
}
```

### Chunk模式（分块提取）

**创建新记录**：
```json
{
  "records": [
    {
      "action": "new",
      "data": {
        "数据标识": "...",
        ...全部28个字段
      }
    }
  ]
}
```

**完善已有记录**：
```json
{
  "records": [
    {
      "action": "enrich",
      "record_index": 1,
      "data": {
        "球头信息.球头-物理性能": "{\"硬度\": \"450 HV\", ...}",
        ...只包含要更新的字段
      }
    }
  ]
}
```
