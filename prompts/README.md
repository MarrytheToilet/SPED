# Prompt 模板说明

本目录包含人工关节材料数据提取系统的所有prompt模板。

## 📁 目录结构

```
prompts/
├── system_prompt.md      # 系统提示词（所有模式共用）
├── modes/
│   ├── skeleton_mode.md  # 骨架识别模式（两阶段-第一阶段）
│   ├── chunk_fill_mode.md # 数据填充模式（两阶段-第二阶段）
│   └── full_mode.md      # 全量提取模式（单次提取）
├── examples/
│   └── full_output.json  # 输出示例
├── prompt.md             # 旧版完整prompt（保留兼容）
└── README.md             # 本文件
```

## 🔄 提取模式

### 1. 两阶段提取（skeleton_fill）- 推荐

适用于长论文（>8000字符），分两阶段提取：

**阶段1 - 骨架识别 (skeleton_mode.md)**
- 输入：论文全文（可能截断/采样）
- 输出：记录骨架JSON（有多少条记录、每条记录的基本特征）
- 目的：确定提取目标

**阶段2 - 数据填充 (chunk_fill_mode.md)**
- 输入：文本块 + 骨架信息
- 输出：每个块提取到的具体数值
- 目的：逐块提取详细数据

### 2. 全量提取（full）

适用于短论文（<8000字符），一次性完整提取：

**单阶段 (full_mode.md)**
- 输入：论文全文
- 输出：完整12表JSON结构
- 目的：快速提取小文档

| 表号 | 表名 | 内容 |
|------|------|------|

| Sheet2 | 内衬基本信息表 | 内衬材料、工艺、添加剂 |
| Sheet3 | 球头基本信息表 | 球头材料、尺寸、工艺 |
| Sheet4 | 配合信息表 | 配合副间隙 |
| Sheet5 | 股骨柄基本信息表 | 股骨柄设计参数 |
| Sheet6 | 内衬物理性能表 | 内衬力学性能 |
| Sheet7 | 球头物理性能表 | 球头力学性能 |
| Sheet8 | 股骨柄物理性能表 | 股骨柄力学性能 |
| Sheet9 | 实验参数 | 摩擦磨损实验条件 |

| Sheet12 | 计算模拟图像表 | 模拟结果图像 |

## 🚫 五条铁律

所有prompt都强调以下规则：

1. **禁止编造** - 论文没写的填 null
2. **禁止转换** - 单位原样保留
3. **保留精度** - 误差范围原样保留
4. **禁止推断** - 不根据图表估算
5. **空值统一** - 用 null，不用空字符串

## 🔧 模板占位符

| 占位符 | 用途 |
|--------|------|
| `{PAPER_CONTENT}` | 论文全文或采样内容 |
| `{CHUNK_CONTENT}` | 当前文本块内容 |
| `{CHUNK_INDEX}` | 当前块序号 |
| `{TOTAL_CHUNKS}` | 总块数 |
| `{SKELETON_INFO}` | 骨架记录信息 |
| `{SKELETON_JSON}` | 完整骨架JSON |
